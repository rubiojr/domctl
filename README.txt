= Xen Cluster Management Tool

* http://domctl.xen-fu.org

== DESCRIPTION:

Simple tool to manage a farm of Xen physical hosts.

It also serves as an example of Pangea xen-api ruby implementation (http://pangea.xen-fu.org)
